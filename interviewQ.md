# Interview Notes

Interviews are recorded here. The answers are summarized for better understanding.


## MO (Dr. Ling Ma)

### 1. What is the job of the administrators?
Admins should balance the workload. If one TA sends plenty of CVs and is chosen by multiple MOs, they can only accept one offer. This means the admin has to control the number of jobs a TA can take.

### 2. How do you currently advertise or post TA positions? How do you want to review and select applicants?
Currently, Excel is used to pass around and input information, which is too complicated. The MO wants a system more like a list of CVs (similar to an interface) where they can easily review candidates and give offers.
