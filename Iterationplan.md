## Iteration Plan

The development follows an **Agile methodology** , with the project divided into four primary iterations to ensure incremental delivery and continuous feedback.

| Iteration | Focus | Expected Outcome |
| :--- | :--- | :--- |
| **Iteration 1** | Requirements, Prototype and user login | Finalized Product Backlog, UI Prototype, Brief Report and start coding by designing code structure and finishing user registration and login function. |
| **Iteration 2** | Core TA & MO Functions | TAs can see jobs and MO can view CVs of TAs. |
| **Iteration 3** | Job and CV upload, admin management | Implementation of Job Creation and CV Upload features, admin Workload Management. |
| **Iteration 4** | Advanced Features | Application Tracking, and AI-powered functions. |

---

###  Key Assessment Milestones
* **22nd March 2026**: First Assessment (Product Backlog, Prototype, and Report).
* **12th April 2026**: Intermediate Assessment (Working Software Iteration 3 Demonstration).
* **24th May 2026**: Final Assessment (Full Software, Video Demo, and Final Report).
